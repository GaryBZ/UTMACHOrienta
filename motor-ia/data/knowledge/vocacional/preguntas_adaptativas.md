# Guia para generar preguntas adaptativas

La IA puede redactar preguntas, pero el objetivo de la pregunta debe venir del motor.

## Reglas

- Hacer una sola pregunta por turno.
- Usar lenguaje claro y cercano.
- No mencionar carreras especificas cuando se este explorando intereses generales.
- No repetir dimensiones ya suficientemente exploradas.
- Evitar preguntas que sugieran una respuesta correcta.
- Incluir opciones cuando el motor necesite datos comparables.
- Permitir texto libre cuando la respuesta pueda ser muy personal.

## Tipos de pregunta

### Opciones

Util para elegir una preferencia principal.

Ejemplo:

¿Que tipo de actividad te resulta mas atractiva?

- Analizar datos o resolver problemas.
- Ayudar directamente a personas.
- Crear contenido visual o comunicativo.
- Organizar proyectos o negocios.

### Multiple

Util cuando varias opciones pueden representar al estudiante.

Ejemplo:

¿Cuales de estas habilidades sientes mas cercanas a ti?

- Comunicar ideas.
- Resolver problemas logicos.
- Observar detalles.
- Crear propuestas visuales.

### Escala

Util para medir intensidad.

Ejemplo:

¿Que tan importante es para ti ayudar directamente a otras personas en tu futuro trabajo?

- 1 - Nada importante
- 2
- 3
- 4
- 5 - Muy importante

### Texto libre

Util para conocer motivaciones o dudas.

Ejemplo:

Si imaginas tu vida universitaria ideal, ¿que te gustaria estar aprendiendo o haciendo?

## Objetivos adaptativos

### Explorar dimension faltante

Si falta entorno, preguntar por el lugar o contexto donde se imagina trabajando.

### Desempatar carreras

Si dos carreras estan cerca, preguntar por una diferencia real entre ellas: datos vs personas, campo vs oficina, salud vs investigacion, creatividad vs gestion.

### Confirmar interes fuerte

Si una afinidad sube muy rapido, preguntar por una actividad concreta relacionada para confirmar.

### Cierre reflexivo

Antes del resultado final, preguntar por valores o expectativas si el perfil esta tecnicamente claro pero falta motivacion personal.
