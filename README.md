# 🎓 UTMACHOrienta — Plataforma de Orientación Vocacional e Inducción Universitaria

## 📌 Descripción del Proyecto

**UTMACHOrienta** es una plataforma web y móvil dirigida a estudiantes de último año de bachillerato que aún no tienen claro qué carrera universitaria elegir en la Universidad Técnica de Machala (UTMACH).

La plataforma busca facilitar la toma de decisiones vocacionales mediante herramientas interactivas como test vocacionales inteligentes, exploración de carreras reales, simulación de clases y exámenes, mapa interactivo del campus y toda la información actualizada del proceso de admisión universitaria.

Es una guía digital completa para acompañar al futuro universitario desde la orientación inicial hasta su ingreso a la universidad.
